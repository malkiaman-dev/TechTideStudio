# 🌊 TechTide Studio

A modern, responsive, and visually engaging digital agency website built with **React**, **Vite**, and **Tailwind CSS**.

TechTide Studio showcases services, portfolio projects, testimonials, team members, and contact details with sleek animations and an intuitive user experience.

---

## 🚀 Overview

The **TechTide Studio** website is designed to create a strong digital presence for a tech agency or creative studio.

It includes:

* Responsive and mobile-friendly design
* Smooth animations and transitions
* Interactive sections and modern layouts
* Service showcase and project portfolio
* Contact and lead generation forms

---

## ✨ Features

### 🎨 Modern UI/UX

* Clean and minimal interface
* Fully responsive on all devices
* Smooth scrolling and section navigation

### 🎬 Animations & Motion Effects

Powered by:

* **Framer Motion**
* **GSAP**

Used for:

* Hero section animations
* Scroll-triggered transitions
* Interactive hover effects

### 📂 Portfolio Showcase

Display projects with:

* Images
* Descriptions
* Technologies used

### 💬 Testimonials

Client reviews and feedback section for trust-building.

### 📩 Contact Integration

Includes:

* Contact forms
* Social links
* Email / phone details

## ⚙️ Installation

Clone the repository:

```bash id="1a5zh0"
git clone <your-repo-url>
cd TechTideStudio
```

Install dependencies:

```bash id="j69q5u"
npm install
```

---

## ▶️ Run Locally

Start the development server:

```bash id="vdf4hh"
npm run dev
```

Build for production:

```bash id="xbhfsq"
npm run build
```

Preview production build:

```bash id="uzx5k0"
npm run preview
```
